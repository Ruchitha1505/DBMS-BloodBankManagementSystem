# 🩸 Blood Bank Management System

A SQL-based Blood Bank Management System developed as a DBMS mini project to manage donor information, patient records, blood inventory, donations, and blood requests efficiently.

---

## 📖 Project Overview

The Blood Bank Management System is a relational database application designed to automate blood bank operations. It helps maintain donor and patient information, manage blood stock, record donations, process blood requests, and generate reports using SQL.

The project demonstrates core DBMS concepts such as relational database design, primary keys, foreign keys, constraints, SQL queries, triggers, and views.

---

## 🎯 Objectives

- Automate blood bank record management.
- Maintain donor and patient information.
- Track blood stock availability.
- Process blood donations and requests.
- Reduce manual errors and paperwork.
- Ensure data integrity using relational databases.

## ⚙️ How to Run

1. Install MySQL Server and MySQL Workbench.
2. Open MySQL Workbench.
3. Execute the SQL files in the following order:
   - schema.sql
   - insert_data.sql
   - triggers.sql
   - views.sql
   - queries.sql
4. Run the SQL queries to test the system.
